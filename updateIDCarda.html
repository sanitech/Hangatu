የመታወቂያ ካርዱን ለማደስ

አውርድ
  • መታወቂያ ካርድ ለማደስ የሚያስፈልጉ መስፈርቶች
    1. መታወቂያ ካርድ በየሁለት አመቱ መታደስ አለበት።
    2. ከሁለት ዓመት በኋላ ያልታደሰ መታወቂያ ካርድ ተቀባይነት አይኖረውም።
    3. መታወቂያ ካርዶችን ለማደስ በከተማው ወረዳ ደረጃ የተገለጸውን ክፍያ(200 ብር) በቴሌ ብር (Merchant ID እና Operator ID ከሚመለከተው አካል ወስደው) በመክፈል ከወረዳው ገቢዎች ክፍል በደረሰኝ ማወራረድ አለበት።
    4. ነዋሪው መታወቂያውን ለማደስ ወደ ከተማው ቀበሌ ሲመጣ በከተማው ውስጥ የመኖርያ ማረጋገጫውን ከታችኛው ቀበሌ የህዝብ አደረጃጀት (ቀጠና ወይም ዞን) ማቅረብ አለበት።
    5. መታወቂያ ወረቀት ለሶስተኛ ወገን የማይሰጥ ቢሆንም በከተማዋ በትምህርት፣ በጤና እና በሌሎች ምክንያቶች ያልኖረ ሰው ከታወቀ አካል የተሰጠ የምስክር ወረቀት ሲያቀርብ መታደስ አለበት።
    6. አመልካቹ በቀጠና ወይም ዞን ቀጠሮ ቀን(ማክሰኞ እና ሃሙስ)2 ፎቶ በመያዝ፣ 200 ብር በቴሌብር(Merchant ID እና operator ID ከሚመለከተው አካል ተቀብለው) በመክፈል ያለፈበትን መታወቂያ እና ከፋይሉ ባለቤት ጋር ያወራረዱትን ደረሰኝ በመያዝ በወረዳው በመገኘት በ5 ደቂቃ ውስጥ የታደሰውን መታወቂያ ይወስዳል።
  • የመታወቂያ ካርድን የመቀየር ፍቃድ መስፈርቶች
    1. የነዋሪው መታወቂያ ከጠፋ፣ ከተበላሸ፣ ከተቃጠለ፣ በውሃ ውስጥ ከጠፋ ወይም ሰነዶቹ ከተሰረዙ እና የነዋሪውን ማንነት በትክክል ማንፀባረቅ ካልቻሉ የመታወቂያ ካርዱን መለወጥ ይፈቀዳል።
    2. በአንቀጽ 10(1) የተመለከተው የመታወቂያ ካርድ መለወጥ የሚፈቀደው የሚከተሉት መስፈርቶች ከተሟሉ ብቻ ነው።
      1. ሀ) የመታወቂያ ካርዱ በክፍለ ከተማው አደረጃጀት ከተረጋገጠ ቀበሌ ማስረጃ ማቅረብ አለባቸው
      2. ለ) የመታወቂያ ካርዱን መጥፋቱን በአቅራቢያው ላለው ፖሊስ ጣቢያ ማሳወቅ እና የድጋፍ ደብዳቤ ማቅረብ አለበት።
      3. ሐ) በዚህ አንቀጽ አንቀጽ 2(ሀ-ለ) የተመለከተው እንደተጠበቀ ሆኖ ይህ መመሪያ በቅጽ ቁጥር. 4 ሙሉ በሙሉ ማቅረብ ያስፈልጋል።
      4. መ) አገልግሎት መስጠት እስከማይችሉበት ደረጃ ድረስ መታወቂያቸው የተሸበሸበ ወይም የተቦረቦረ ወይም የተቧጨረባቸው ሰዎች የድሮ መታወቂያቸውን በአካል በማቅረብ የመታወቂያ ካርድ ይለዋወጣሉ።
      5. ሠ) ለቀበሌ የሚፈልገውን መታወቂያ ለማደስ የሚያስፈልገውን ክፍያ(200 ብር) በቴሌ ብር (Merchant ID እና Operator ID ከሚመለከተው አካል በመቀበል) በመክፈል ከወረዳው መዘጋጃ ክፍል በደረሰኝ ማወራረድ አለበት።
      6. ረ) አመልካቹ በቀጠና ቀጠሮ ቀን በወረዳው ከፋይሉ ባለቤት ጋር በመገኘት በ5 ደቂቃ ቅያሪውን መታወቂያ ይወስዳሉ።
