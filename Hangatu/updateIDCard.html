Waraqaa eenyummaa haaromsuuf

Buusuu

-ulaagaa duraa፦

  • Waraqaa Fenyummaa Haaromsuuf Ulaagaalee Barbaachisaan.
    1. Waraqaan eenyummaa turtii waggaa lamaan haaromfamu qaba.
    2. Waraqaan eenyummaa turtii waggaa lamaan booda hin haaromfamne fudhatama hin qabaatu.
    3. Haaromsa waraqaa eenyummaatiif kaffaltii magaalaadhaan labsame sadarkaa aanaa magaalatti kaffaluun. kaffaltii barbachisu (Qr.200) dursee telebirrin (Merchent ID fi operator ID qamaa dhimmi ilalatuu irra fudhachuun kafaludhaan nagahee wajjira galii aanaa hangaatuutti busuun tajaajilamu.
    4. Jiraataan waraqaa eenyummaa haaromsuudhaaf gara aanaa magaalaatti yoo dhufu, magaalicha keessa jiraachaa turuu gurmaa'iinsa uummataa aanaa qaxana(zooni) gadiirraa ragaa dhiyeessuu qaba
    5. Keewwata kana 4(4) jalatti kan tumame jiraatus, sababa barnootaan, fayyaa fi kan birootiin, magaalaa keessa jiraachaa kan hin turre yoo ta'e ragaa qaama beekamtii qabuun kenname yoo dhiyeeffatan kan haaromfamu ta'a.
    6. Namni lyyataan gafaa guyyaa kibxataa fi kamisaa waraqaa enyumaa yeroon irra darbe qarshi(200qr) kafaluudhaan haromsuu fi waraqa enyumaa dhufee suraa tokkoo qabu belama qaxana(zoonii) abbaa fayyila waliin Naga’ee qabachudhaan aanaattii argamuudhaan waraqa enyumaa haaromfamee daqiqaa shanin(5) ni kenamaa.
  • Ulaagaalee Jijjiirraan Waraqaa Eenyummaa Ittiin hayyamamu.
    1. Waraqaan eenyummaa jiraataa yoo bade, biinxame, ibiddaan yoo gubate, bishaanitti yoo bade ykn rigataan barreeffamonni haqamanii eenyummaa jiraataa sirriitti ibsuu yoo hin dandeenne, jijjiirraan waraqaa eenyummaa ni hayyamama.
    2. Jijjiirraan waraqaa eenyummaa Keewwata kana 10(1) jalatti tumame kan hayyamamu ulaagaalee armaan gadii guutee yoo dhiyaate qofa.
      1. Gurmaa'iinsaa aanaa gadiin waraqaan eenyummaa jalaa mirkanaa'ee booda aanaa magaalaaf ragaa dhiyeessuu qabu
      2. Waraqaan eenyummaa jalaa baduu isaa Waajjira Poolisii dhiyoo isaatti arga beeksisuun xalayaa deeggarsa dhiyyeeffachuu qaba.
      3. Keewwata kana 2(a-b) jalatti kan tumame akkuma jirutti ta'ee, qajeelfama kana uunkaa lakk. 4 irratti agarsiifame guutuun dhiyyeessuun irraa eegama.
      4. Namoonni waraqaan eenyummaa isaanii jala biinxamee ykn sukkummaadhaan ykn rigataan sadarkaa tajaajila kennuu hin dandeenyee irra gahe, jijjiirraa waraqaa eenyummaa kan argatan waraqaa eenyummaa moofaa qaaman argamuudhaan galmeen isaanii ilaalame kan jijjiiramuuf ta'a.
      5. Kaffaltii haaromsaa fi jijjiirraa waraqaa eenyummaatiif aanaa magaalaatiin gaafatamu jiraatan kaffaltii barbachisu(200 QR)dursee telebirrin dursee telebirrin(mMorchent ID fi operator ID qamaa dhimmi ilalatuu irra fudhachuun kafaludhaan nagahee wajjira galii aanaa hangaatuutti busuu fi kafaludhaa raawwachuu qaba.
      6. Namni lyyataan gaf guyyaa belama qaxana abbaa fayyila waliin Naga’ee qabachudhaan aanaattii argamuudhaan waraqa enyumaa jijiiraa daqiqaa shanin(5) ni kenamaa.
